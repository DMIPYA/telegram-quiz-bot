# 🧠 Telegram Quiz Bot

Тематическая викторина прямо в Telegram. Пользователь выбирает категорию, проходит раунд из 5 вопросов с 4 вариантами ответа, получает баллы в зависимости от сложности и видит пояснение к каждому ответу. Статистика сохраняется между сессиями, по проценту правильных ответов присваивается уровень.

Бот на русском языке, интерфейс целиком на inline-кнопках с эмодзи.

## ✨ Возможности

- 🚀 Команда `/start` — приветствие и главное меню.
- 🎮 **Начать викторину** — выбор категории (или «Любая»), раунд из 5 вопросов с вариантами в 2 колонки.
- ✅ Мгновенная подсветка ответа: зелёная — верно, красная — неверно.
- 💡 Пояснение к каждому вопросу после ответа.
- ⏭️ Кнопка «Дальше» — пользователь сам контролирует темп.
- 🏆 Итог раунда: число правильных ответов и набранные баллы. Дальше — «Ещё раунд» или «В главное меню».
- 📊 **Мой счёт** — статистика: отвечено вопросов, процент правильных, текущий уровень, сумма баллов, сыграно раундов.
- ❓ **Помощь** — инструкция по использованию.
- 🤖 Команда `/generate` — генерация новых вопросов через NVIDIA NIM (только для админов).
- 💾 Статистика хранится в `stats.json`, сохраняется между перезапусками бота.

## 🎯 Категории и вопросы

База вопросов — `questions.json` (массив объектов). Каждый вопрос содержит поля:

| Поле | Описание |
|------|----------|
| `text` | Текст вопроса |
| `options` | Массив из 4 вариантов ответа |
| `correct_option_index` | Индекс правильного ответа (0–3) |
| `explanation` | Пояснение к ответу |
| `difficulty` | Сложность: `easy`, `medium`, `hard` |
| `category` | Категория (одна из 12) |

12 категорий: География, История, Наука, Кино, Литература, Музыка, Спорт, Технологии, Искусство, Природа, Еда, Космос.

> Сейчас в базе **164 вопроса**. Категория доступна в меню, если в ней ≥ 5 вопросов.

## 🏅 Уровни

Уровень пересчитывается от общей накопленной статистики после каждого раунда:

| Процент правильных | Уровень |
|--------------------|---------|
| < 50% | 🌱 Начинающий |
| 50–74% | 🔍 Любознательный |
| 75–89% | 📚 Знаток |
| 90%+ | 🏆 Эксперт |

## ⚖️ Очки

За правильный ответ: `easy` = 1 балл, `medium` = 2, `hard` = 3.

## 🛠️ Технологии

- **Python 3.11+**
- `python-telegram-bot` 20.x (async)
- `python-dotenv` — переменные окружения
- `openai` SDK — обращение к NVIDIA NIM API
- Хранение: JSON-файлы (`questions.json`, `stats.json`)

## ⚙️ Установка

### 1. Клонирование

```bash
git clone https://github.com/DMIPYA/telegram-quiz-bot.git
cd telegram-quiz-bot
```

### 2. Зависимости

```bash
pip install -r requirements.txt
```

### 3. Переменные окружения

Скопируйте `.env.example` в `.env` и заполните:

```bash
cp .env.example .env
```

| Переменная | Обязательно | Описание |
|------------|-------------|----------|
| `BOT_TOKEN` | ✅ | Токен бота от [@BotFather](https://t.me/BotFather) |
| `NVIDIA_API_KEY` | ✅ | Ключ NVIDIA NIM (для `/generate`) |
| `NVIDIA_BASE_URL` | ➖ | Базовый URL NVIDIA NIM (по умолчанию `https://integrate.api.nvidia.com/v1`) |
| `NVIDIA_MODEL` | ➖ | Модель NIM (по умолчанию `meta/llama-3.1-70b-instruct`) |
| `ADMIN_USER_IDS` | ➖ | Telegram `user_id` админов через запятую — кому доступна `/generate` |

> Команда `/start` работает без `NVIDIA_API_KEY`, но бот стартаует только при наличии ключа (проверка при запуске). Задайте `ADMIN_USER_IDS`, чтобы разрешить генерацию вопросов.

### 4. Запуск

```bash
python bot.py
```

## 🤖 Генерация вопросов (`/generate`)

Команда доступна только пользователям из `ADMIN_USER_IDS`. Бот обращается к NVIDIA NIM API (через `openai` SDK), генерирует по 3 вопроса на категорию, валидирует структуру и добавляет в `questions.json`.

```bash
/generate
```

Пример ответа: `✅ Сгенерировано N новых вопросов! 📂 Категории: ...`

## 🔒 Безопасность

- `.env` исключён из git через `.gitignore` — токен и ключи не попадают в репозиторий.
- Команда `/generate` защищена списком админов (`ADMIN_USER_IDS`).
- Получите `BOT_TOKEN` только у официального [@BotFather](https://t.me/BotFather).
- Не публикуйте `NVIDIA_API_KEY` и `BOT_TOKEN`. При компрометации — перевыпустите.
