# Статистика и уровни

Status: ready-for-agent

## What to build

- Создать `stats.json` для персистентного хранения статистики по `user_id`
- Добавить функции: `load_stats()`, `save_stats()`, `update_stats(user_id, correct, total_questions, points)`
- После каждого раунда сохранять статистику в `stats.json`
- Экран «Мой счёт»:
  - Всего вопросов отвечено
  - Правильных ответов
  - Процент правильных
  - Всего баллов
  - Текущий уровень
- Уровни от общего процента правильных:
  - < 50% — Начинающий 🌱
  - 50-74% — Любознательный 🔍
  - 75-89% — Знаток 📚
  - 90%+ — Эксперт 🏆
- Уровень пересчитывается после каждого раунда и при просмотре статистики

Структура stats.json:
```json
{
  "123456789": {
    "total_questions": 15,
    "correct_answers": 11,
    "total_points": 28,
    "games_played": 3
  }
}
```

## Acceptance criteria

- [ ] `stats.json` создаётся автоматически при первом сохранении
- [ ] После раунда статистика сохраняется в файл
- [ ] После перезапуска бота статистика загружается из файла
- [ ] Кнопка «Мой счёт» показывает: вопросы, правильные, %, баллы, уровень
- [ ] Уровень правильно рассчитывается по проценту
- [ ] Эмодзи в каждом уровне
