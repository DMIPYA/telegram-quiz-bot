# База вопросов

Status: ready-for-agent

## What to build

- Создать `questions.json` с массивом из 60+ вопросов (минимум по 5 на каждую из 12 категорий)
- В `bot.py` добавить загрузку вопросов из JSON при старте
- Реализовать функцию `get_random_questions(category, count)` — выборка N случайных вопросов без повторов
- Реализовать функцию `get_categories()` — возвращает список категорий, где ≥ 5 вопросов
- Реализовать `get_category_count(category)` — количество вопросов в категории

Формат вопроса:
```json
{
  "text": "Вопрос?",
  "options": ["А", "Б", "В", "Г"],
  "correct_option_index": 1,
  "explanation": "Пояснение",
  "difficulty": "easy",
  "category": "Наука"
}
```

12 категорий: Астрономия, История, География, Древний мир, История России, Кино, Литература, Музыка, Наука, Спорт, Химия, Всемирная история.

Сложности распределить примерно: 1/3 easy, 1/3 medium, 1/3 hard.

## Acceptance criteria

- [ ] `questions.json` содержит ≥ 60 вопросов, ≥ 5 на категорию
- [ ] Все вопросы имеют корректный формат (4 options, correct_option_index 0-3)
- [ ] Бот загружает вопросы при старте без ошибок
- [ ] `get_random_questions("Наука", 5)` возвращает 5 разных вопросов категории «Наука»
- [ ] `get_categories()` возвращает список из 12 категорий
- [ ] Если в категории < 5 вопросов — она не включается в `get_categories()`
