# Quiz TG Bot — Telegram-бот-викторина

## Язык

Русский — весь UI, названия кнопок, сообщения, пояснения, категории.

## Стек

- **Python**, библиотека `python-telegram-bot`
- `python-dotenv` для токена из `.env`
- Хранение вопросов: `questions.json`
- Хранение статистики: `stats.json` (персистентно)

## Структура файлов

```
/
├── bot.py              # основной файл
├── questions.json      # база вопросов
├── stats.json          # статистика игроков
├── .env                # токен бота
└── requirements.txt
```

## Формат вопроса (questions.json)

```json
{
  "text": "Вопрос?",
  "options": ["А", "Б", "В", "Г"],
  "correct_option_index": 1,
  "explanation": "Пояснение",
  "difficulty": "easy",
  "category": "Наука"
}
```

Всегда 4 варианта ответа. `correct_option_index` от 0 до 3.

## Категории (12)

Астрономия, История, География, Древний мир, История России, Кино, Литература, Музыка, Наука, Спорт, Химия, Всемирная история.

Вопросов 50+, по 4-5 на категорию.

Раунд доступен для категории только если в ней ≥ 5 вопросов. «Любая» выбирает из всех вопросов без ограничения.

## Команды и меню

- `/start` — приветствие + главное меню (inline-кнопки)
- **«Начать викторину»** → выбор категории (включая «Любая») → раунд 5 вопросов
- **«Мой счёт»** — статистика игрока
- **«Помощь»** — инструкция

## Flow викторины

1. Пользователь выбирает категорию
2. Бот выбирает 5 случайных вопросов (без повторов в раунде)
3. Показывает вопрос с 4 вариантами в 2 колонки (inline-кнопки)
4. Пользователь нажимает ответ → бот показывает:
   - правильно/неправильно (зелёная/красная подсветка)
   - пояснение
   - кнопка «Дальше»
5. После 5-го вопроса — итог раунда (сколько правильных / баллов)
6. Кнопки: «Ещё раунд» / «В главное меню»

## Система очков

- easy = 1 балл
- medium = 2 балла
- hard = 3 балла

## Уровни (по общей накопленной статистике, % правильных)

| Процент | Уровень |
|---------|---------|
| < 50% | Начинающий |
| 50-74% | Любознательный |
| 75-89% | Знаток |
| 90%+ | Эксперт |

Пересчитывается после каждого раунда.
