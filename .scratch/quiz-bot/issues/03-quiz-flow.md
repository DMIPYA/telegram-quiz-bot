# Flow викторины

Status: ready-for-agent

## What to build

Полный игровой цикл:

1. Кнопка «Начать викторину» в главном меню → показывает выбор категории (inline-кнопки в 2 ряда)
2. Категории из `get_categories()` + кнопка «Любая»
3. Выбор категории → раунд из 5 вопросов:
   - Показ вопроса + 4 варианта ответа (inline-кнопки в 2 колонки по 2)
   - После ответа: подсветка (✅ правильно / ❌ неправильно) + пояснение + кнопка «Дальше»
   - После 5-го вопроса: итог раунда (сколько правильных из 5, сколько баллов)
   - Кнопки после итога: «Ещё раунд» / «В главное меню»
4. Очки: easy=1, medium=2, hard=3
5. Статистика пока только в памяти (dict по user_id), без сохранения в stats.json

Использовать `CallbackContext.user_data` или глобальный dict для хранения состояния раунда и временной статистики. Ответы обрабатывать через `CallbackQueryHandler`. Для разметки кнопок в 2 колонки — `InlineKeyboardMarkup` с `row_width=2`.

## Acceptance criteria

- [ ] «Начать викторину» показывает inline-список категорий + «Любая»
- [ ] Выбор категории запускает раунд из 5 вопросов
- [ ] Варианты ответов расположены в 2 колонки
- [ ] После ответа: зелёная/красная отметка + пояснение + «Дальше»
- [ ] После 5-го вопроса: итог с правильными/баллами
- [ ] Кнопки «Ещё раунд» и «В главное меню» работают
- [ ] Очки начисляются по сложности
- [ ] Возврат в меню после раунда
