# Помощь и полировка

Status: ready-for-agent

## What to build

- Экран «Помощь»: inline-сообщение с инструкцией по использованию бота
- Проверить и добавить эмодзи во все сообщения бота (приветствие, меню, кнопки, результаты, итоги)
- Финальная проверка всех связок: меню → викторина → статистика → помощь
- Базовая обработка ошибок (если questions.json повреждён, если stats.json повреждён, если пользователь нажал кнопку из устаревшего сообщения)

Текст помощи:
```
🧠 Как играть в викторину?

1. Нажми «Начать викторину» в главном меню
2. Выбери категорию (или «Любая»)
3. Ответь на 5 вопросов подряд
4. Получи очки за правильные ответы
5. Следи за прогрессом в «Мой счёт»

Сложность вопросов:
🟢 Лёгкий — 1 балл
🟡 Средний — 2 балла
🔴 Сложный — 3 балла

Уровни:
🌱 Начинающий — < 50%
🔍 Любознательный — 50-74%
📚 Знаток — 75-89%
🏆 Эксперт — 90%+

Вопросы добавляются регулярно. Удачи! 🍀
```

## Acceptance criteria

- [ ] «Помощь» показывает инструкцию
- [ ] Эмодзи присутствуют во всех сообщениях бота
- [ ] Бот не падает при повреждённом questions.json (выводит readable ошибку)
- [ ] Бот не падает при повреждённом stats.json (начинает с чистого файла)
- [ ] Нажатие кнопки из старого сообщения не ломает бота (обработка через `answer_callback_query`)
